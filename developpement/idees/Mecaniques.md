## Fusil formique

| Fusil Formique  | 
| :--------------- |
| Pause : Passer 2  |
| Distance  |
| Défausser 1 | 
| Blesser 7 | 
| Reprendre la carte en main| 

Le fusil formique est une vieille carte. La mécanique de "Passer" n'existe plus, mais dans ce cas là pourrait être remplacée par "l'adversaire a : Rejouer / Rejouer".

ça laisse à l'adversaire le temps de se placer au corps à corps et empêcher que les étapes ne se résolvent. Si elle se résolvent, le.a joueur.se reprendra la carte dans sa main.

On pourrait ainsi introduire le mot clef "Reprendre" ou "Récupérer". Une version plus actuelle serait donc :

| Fusil Formique  | 
| :--------------- |
| Défausser 1 | 
| L'adversaire a : Rejouer / Rejouer |

## Situels

Les situels sont des cartes qui on un effet dans un certain rayon autour d'un point. Ca permet d'avoir une meilleure exploitation du terrain.

Ca introduit les mot clefs "marquer" et "Rayon". Mais la plupart des cartes sont obsolète. Un exemple d'idée actualisé

| Situel d'épuisement  | 
| :--------------- |
| Marquer | 
| Rayon 2 : jouer une technique : Se Blesser 1|
| Rayon 1 : Pause 1| 

Marquer revient à pauser un jeton sur une case adjacente. Ensuite tout les rayons évoqués font référence à se jeton. Donc ici si un.e joueur.se joue une technique dans un rayon de 2 cases autour du jeton, il se blessera de 1.
