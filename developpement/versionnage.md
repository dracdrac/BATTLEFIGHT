# Identification des cartes

Le titre d'une carte n'est pas toujours une information suffisante pour identifier une carte. 

L'identification précise des cartes peut se faire grâce à un identifiant inscrit en bas à gauche de la carte.

## Anatomie de l'identifiant


L'identifiant est inscrit sur deux lignes. Par exemple, l'identifiant peut ressembler à ça :

    I 00
    1 ST

La première ligne est suffisante pour identifier la carte en terme de règles et de gameplay. Elle contient :

- Le numéro de saison
- Le numéro de carte

La deuxième ligne donne des information supplémentaires pour identifier la carte en terme de collection. Elle contient :

- Le numéro d'édition
- Le code de traitement

## Le numéro de saison

Le numéro de saison est inscrit en chiffre romains.

Une saison est un ensemble de 100 cartes publiées à un même moment et en même nombre d'exemplaires qui forment un tout cohérent en terme de mécaniques de jeu et de lore.

## Le numéro de carte

Le numéro de carte est écrit en deux chiffres. De 00 à 99, il désigne la place de la carte dans la saison.

## Le numéro d'édition

Le numéro d'édition est un numéro incrémenté à chaque fois que les cartes de la saison sont réimprimées.

La première édition officielle d'une saison porte le numéro 1.

Avant la publication de l'édition 1, on publie une édition 0, c'est la dernière version de travail, conçue pour être la plus proche possible de l'édition 1, elle est distribuée gratuitement et sans garantie à des fins de test ou de promotion.

En interne, on parle d'éditions alpha pour désigner les versions de travail qui précèdent l'édition 0.

## Le code du traitement

Le code du traitement est un code de deux lettres majuscule qui identifie le traitement esthétique de la carte.

Par exemple "ST" pour le traitement Standard, "BL" pour BorderLess, "GA" pour Guest Art, etc.
